# Kalkulačka rampy pre vozidlá

Jednoduchý webový nástroj na výpočet dĺžky šikmej rampy pre vozidlá podľa STN 73 6058.

## Funkcie

- **4 typy rámp**: vnútorná priamočiara (14 %), vnútorná krivočiara (13 %), vonkajšia hlavná (10 %), vyrovnávacia/polo (17 %)
- **2 kategórie vozidiel**: všetky vozidlá (rv1 = 12 m, rv2 = 20 m) / autobusy (rv1 = 20 m, rv2 = 75 m)
- **2 typy nábehov**: výškové oblúky alebo alternatívne riešenie s 5 % medzisklonom
- **Real-time výpočet**: celková dĺžka, horný nábeh, priamy úsek, dolný nábeh (dĺžka + výška)
- **SVG vizualizácia**: pozdĺžny profil rampy s farebným rozkladom segmentov

## Použitie

Otvorte `index.html` v prehliadači. Žiadne závislosti, žiadny build.

## Norma

Výpočty vychádzajú z STN 73 6058 — Jednotlivé, radové a hromadné garáže (kapitola 8.2 Šikmé rampy pre vozidlá).
